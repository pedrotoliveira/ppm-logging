============
#What is PPM Logging
- A Simplified Facade API to Log4J 2;
- A Fluent Log API;
- A Reactive Log Method;
- A Convention Configuration Method of Log4J.

#What PPM Logging is NOT.
- SL4J (For project reasons i decided not use SL4J).
- A Standalone Logging Engine
