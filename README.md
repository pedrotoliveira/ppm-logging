## What is PPM Logging
- A Simplified Facade API to Log4J2;
- A Fluent Log API;
- A Convention Configuration Method to Log4J.

## What PPM Logging is NOT.
- SL4J (For project reasons i decided do not use SL4J).
- A Standalone Logging Engine
